### 2. `display: grid;` (Grid Layout) The Grid Layout is a two-dimensional layout system, meaning it can handle both rows and columns. #### Key Concepts of Grid - Grid Container: The parent element that has `display: grid;`. - Grid Items: The child elements inside the grid container. #### Grid Container Properties 1. `grid-template-columns`: - Defines the columns of the grid. - Syntax: `grid-template-columns: 100px 200px;` or `repeat(3, 1fr);`. 2. `grid-template-rows`: - Defines the rows of the grid. - Syntax: `grid-template-rows: 100px 200px;` or `repeat(2, 1fr);`. 3. `gap`: - Specifies the gap between grid items. - Syntax: `gap: 10px;` or `grid-gap: 10px;`. 4. `justify-items` and `align-items`: - Defines the default alignment for items within their grid area. - Values: `start`, `end`, `center`, `stretch`. #### Example of Grid Layout:
